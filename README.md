#Student Registration Status Check.

## Purpose of the project.

This specific project is used to demonstarte how one problem, which was Student Registration Status Check, could be solved using various programming languages.

## How to run the Python version.

</>bash
cd pyhton
Python.py

## How to run the C version.

</>bash
cd c
gcc registration.c -o regristration
./registraion

## How to run the Java version.

</>bash
cd java
javac RegistrationChecker.javac
java RegistraionChecker

## How to run the JavaScript version.

</>bash
cd javascript
node registration.js
