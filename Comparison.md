# Language Comparison for the Student Registration Checker

## Overview

This is used to compare how 4 different programming languages; Python, Java, JavaScript and C will implement a Student Regristation Status Checker.

## Comparison

## Python

- Simple and reliable to use
- Uses dymanic tpying
- User friendly espesialy for beginners
- Fast and easy to implement

## Java

- More structered making it challenging for beginners
- Uses static typing
- Has many syntax rules
- Most convinient for enterprise systems

## JavaScript

- Flexible and widely used
- Uses dynamic typing
- Runs in browser
- Often used for web development by beginners

## C

- Low-level and fast
- Uses static typing
- Compiled language
- Complex language often for large scale systems

## Key Differences

| Feature     | Python          | Java          | JavaScript      | C             |
| ----------- | --------------- | ------------- | --------------- | ------------- |
| Typing      | Dynamic         | Static        | Dynamic         | Static        |
| Compilation | Interprated(No) | Compiled(Yes) | Interprated(No) | Compiled(Yes) |
| Ease of Use | High            | Medium        | High            | Low           |
| Performance | Medium          | High          | Medium          | Very High     |

## Conclusion

Python is best for simplicity, Java for structure, JavaScript for websites, and C for performance. Either of the programing languages can be used depending on the users prefrence and task to be completed.
